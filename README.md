# 🚀 IT Incident Management Automation (Power Automate)

> ⚠️ Nota: Este proyecto es una representación demostrativa basada en una solución real. No contiene información sensible ni datos confidenciales.

## 📌 Descripción

Automatización del registro, seguimiento y análisis de incidentes TI mediante flujos en Power Automate.

El objetivo es reducir tareas manuales, centralizar la información y mejorar la trazabilidad de eventos para la gestión de riesgos.

## 🛠️ Tecnologías

* Power Automate
* Microsoft Teams
* Excel
* Outlook

## ⚙️ Funcionalidades

* 📥 Registro automático de incidentes desde múltiples canales
* 📊 Centralización de datos en Excel
* ⏱️ Cálculo automático de tiempos de interrupción
* 🧠 Clasificación por categorías de incidentes
* 🔍 Seguimiento y trazabilidad
